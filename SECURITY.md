# Security Policy

## Supported Versions

The following versions of KIRO Tetris AI are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please report it to us by creating a GitHub Security Advisory or by contacting the maintainers directly.

Please do not report security vulnerabilities via public GitHub issues.

### Our Process

Once a report is received, we will:
1. Acknowledge the report within 48 hours.
2. Investigate the issue and confirm its validity.
3. Work on a fix and release a security update as soon as possible.
4. Provide credit to the reporter in the security advisory if requested.
